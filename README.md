# My Interactive Web Dashboard 🚀

This is a static website hosted on **GitHub Pages** that allows for basic user interaction and Google Spreadsheet integration.

## 🛠 Features
* **1st Button:** Plays a specific YouTube video.
* **Textbox:** Input area for custom messages.
* **2nd Button:** Shows the message in a **Large Font Size (50px)**.
* **3rd Button:** Quick link to the master Google Spreadsheet.
* **Embedded Sheet:** Live view of student data/survey results at the bottom.

## 📂 How to Upload or Update Files
To add new files (like images or data sheets) to this project:

1.  Go to your repository: [https://github.com/MarukuShea/MyWeb](https://github.com/MarukuShea/MyWeb)
2.  Click the **Add file** button (the `+` icon) at the top right-hand corner of the file list.
3.  Choose **Create new file** to write code directly, or **Upload files** to drag and drop from your Windows 10 computer.
4.  Always click the green **Commit changes** button at the bottom to save your work.

## ⚙️ How to Change Content
If you need to update the YouTube video or the Spreadsheet link:
* Click on `index.html`.
* Click the **Pencil Icon** (Edit this file).
* Change the `videoId` or the `iframe src` link.
* **Commit changes** to see the update live on your website.

## 🚀 How to Launch the Website (First Time Setup)
If the website is not appearing at your link yet, follow these steps to "turn it on":

1.  Go to your repository: [https://github.com/MarukuShea/MyWeb](https://github.com/MarukuShea/MyWeb)
2.  Click the **Settings** tab (the gear icon ⚙️) at the very top of the page.
3.  On the left-hand sidebar, click **Pages**.
4.  Under **Build and deployment** > **Branch**:
    * Click the dropdown that says `None`.
    * Select `main` (or `master`).
    * Ensure the folder is set to `/(root)`.
5.  Click **Save**.
6.  **Wait 1-2 minutes.** Refresh the page until a bar appears at the top saying: *"Your site is live at..."*
7.  Click that link to view your site!
